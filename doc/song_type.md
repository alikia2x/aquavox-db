# Song Type

The `type` column used in the PostgreSQL database.

| Type | Description            |
|------|------------------------|
| 0    | Other                  |
| 1    | Original               |
| 2    | Cover                  |
| 3    | Remix                  |
| 4    | Instrumental           |
| 100  | AI tagged: Unrelated   |
| 101  | AI tagged: Original    |
| 102  | AI tagged: Cover/Remix |
